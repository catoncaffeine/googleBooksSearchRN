## Pre-requisite:

1. node 8.3+
2. watchman
3. react-native cli
4. Xcode if iOS
5. Android Studio if Android

## Initial Set Up
Run command
```
yarn
```
or
```
npm install
```

## Run in iOS Simulator
Run command:
```
react-native run-ios
```

## Run in Android Emulator

1. open Android Studio
2. Set up a device in AVD manager
3. Run commands:
```
react-native run-android
```
